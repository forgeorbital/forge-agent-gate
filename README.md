# forge-agent-gate

> ## This package is retired
>
> Forge Orbital has moved to an enterprise-only model. `forge-agent-gate`, the
> self-serve developer package, is no longer maintained or supported, and the
> hosted service behind it has been retired. Copies already installed will keep
> running, but there will be no further releases, fixes, or support.
>
> Forge Orbital's decision, audit, and replay capability for AI-agent actions is
> now delivered as an enterprise, server-to-server integration, not a local
> developer tool.
>
> For enterprise access, contact **joseph@forgeorbital.com**.
